# 🚀 CodeAlpha Machine Learning Projects

This repository contains multiple machine learning projects completed as part of the **CodeAlpha Internship Program**.  
Each project demonstrates end-to-end skills including data preprocessing, model building, evaluation, and deployment.

## 📂 Projects Included

### 🌸 1. Iris Classification
- Built a classification model to identify iris flower species.
- Applied data preprocessing and supervised learning techniques.
- Evaluated model accuracy.

- Predicted sales based on TV, Radio, and Newspaper advertising budgets.

### 🚗 3. Car Price Prediction
- Predicted car selling prices using regression techniques.
- Performed feature engineering and data cleaning.
- Evaluated model performance using regression metrics.

📁 Folder: `CodeAlpha_car_prediction`

### 👩‍💼 4. Employment Prediction Project
- Built a machine learning model related to employment analysis/prediction.
- Applied preprocessing, model training, and evaluation.
